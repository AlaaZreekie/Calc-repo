# Calc-repo
First versio of a calculator API 
this API get an expersion from the user _the body _ and return the result 

for example:
  in the bode :
                expression = 1+2+3
  returns:      the ans is 6

notes:
  1. Write the expresion with out spaces    //1 + 2 + 3 not valid right now
  2. The operaton that are valid are (+, -, /, *)
